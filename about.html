About This Project

Module 6WCM0029 — Computer Science Final Year Project

What This Project Does

This project builds an AI system that predicts lung disease risk from patient health data. Two ML models — Logistic Regression and Decision Tree — are trained on a Kaggle dataset and integrated into a Django web app.

The user enters 15 health features (age, smoking, symptoms etc.) and the system gives a risk label — HIGH RISK or LOW RISK — with probability scores from each model.

How It Was Built — Step by Step

  • Step 1: DatasetLung Cancer Survey dataset from Kaggle — 1,000 records, 15 features.
  • Step 2: PreprocessingCleaned data, encoded labels, applied StandardScaler, 80/20 train-test split.
  • Step 3: Train ModelsTrained Logistic Regression and Decision Tree using Python and Scikit-learn.
  • Step 4: EvaluateMeasured accuracy, precision, recall, F1, and 5-fold cross-validation.
  • Step 5: Save ModelsSaved as .pkl files with Joblib so Django can load them at startup.
  • Step 6: Web AppUser enters data → Django calls model → result displayed on screen.

⚖️ Ethics

Only public, anonymised datasets used. No patient data stored. This is a decision-support tool, not a replacement for medical professionals. Follows BCS Code of Conduct principles.
